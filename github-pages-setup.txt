# GitHub Pages 部署步驟

1. 首先創建以下檔案結構：

```
test/
├── public/
│   └── index.html
├── src/
│   ├── components/
│   │   └── ui/
│   │       ├── button.jsx
│   │       ├── card.jsx
│   │       └── input.jsx
│   ├── App.jsx
│   ├── index.jsx
│   └── DiceGame.jsx
├── .gitignore
├── package.json
├── vite.config.js
└── README.md
```

2. 在各個檔案中放入以下內容：

`public/index.html`:
```html
<!DOCTYPE html>
<html lang="zh-TW">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>擲骰子遊戲</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/index.jsx"></script>
  </body>
</html>
```

`package.json`:
```json
{
  "name": "test",
  "private": true,
  "version": "0.0.0",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview",
    "deploy": "gh-pages -d dist"
  },
  "dependencies": {
    "react": "^18.2.0",
    "react-dom": "^18.2.0",
    "@radix-ui/react-slot": "^1.0.2",
    "class-variance-authority": "^0.7.0",
    "clsx": "^2.0.0",
    "tailwind-merge": "^2.0.0",
    "tailwindcss": "^3.3.0"
  },
  "devDependencies": {
    "@types/react": "^18.2.15",
    "@types/react-dom": "^18.2.7",
    "@vitejs/plugin-react": "^4.0.3",
    "autoprefixer": "^10.4.14",
    "gh-pages": "^6.0.0",
    "postcss": "^8.4.27",
    "vite": "^4.4.5"
  }
}
```

`vite.config.js`:
```javascript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  base: '/test/', // 這裡要改成你的 repository 名稱
})
```

`.gitignore`:
```
node_modules
dist
dist-ssr
*.local
```

3. 部署步驟：

```bash
# 1. 安裝依賴
npm install

# 2. 構建專案
npm run build

# 3. 部署到 GitHub Pages
npm run deploy
```

4. GitHub 配置：

- 到 GitHub 倉庫設置 (Settings)
- 找到 Pages 部分
- Source 選擇 `gh-pages` 分支
- 儲存設置
